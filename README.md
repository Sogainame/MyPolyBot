# 🤖 Polymarket Arbitrage Scanner v0.1

Сканер арбитражных возможностей на Polymarket.
Находит расхождения в ценах и оповещает через Telegram.

## Что ищет

| Тип | Описание | Пример |
|-----|----------|--------|
| **intra_market** | YES + NO < $1.00 в бинарном рынке | YES=0.45 + NO=0.52 = 0.97 → 3% профит |
| **intra_event** | Сумма YES > $1.00 в negRisk-событии (перегрет) | 3 кандидата, сумма YES = 1.05 → split $1 → sell all → 5% профит |

### Важно: что НЕ является арбитражем
- Сумма YES < 1.00 в negRisk — это **нормально**, остаток = вероятность "Other" (кого нет в списке)
- Multi-market событие без negRisk (напр. "Gold > $3000" и "Gold > $3100") — рынки не взаимоисключающие

## Быстрый старт

```bash
# 1. Клонируй / скопируй файлы
cd polymarket-arb-bot

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Тестовый запуск (один скан)
python main.py

# 4. Бесконечный цикл
python main.py --loop

# 5. Цикл + Telegram-уведомления
cp .env.example .env
# Заполни TELEGRAM_BOT_TOKEN и TELEGRAM_CHAT_ID
python main.py --loop --tg
```

## Настройка Telegram

1. Открой `@BotFather` → `/newbot` → получи токен
2. Напиши своему боту `/start`
3. Открой `https://api.telegram.org/bot<ТОКЕН>/getUpdates` → найди `chat_id`
4. Запиши в `.env`

## Конфигурация (config.py)

| Параметр | Значение | Описание |
|----------|----------|----------|
| `SCAN_INTERVAL_SEC` | 30 | Интервал между сканами |
| `MIN_PROFIT_PCT` | 1.0 | Мин. чистая прибыль (%) |
| `MIN_LIQUIDITY` | 5000 | Мин. ликвидность рынка ($) |
| `WINNER_FEE_PCT` | 2.0 | Комиссия Polymarket |
